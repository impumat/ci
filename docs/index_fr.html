Foire aux questions

IMPUMAT fonctionne-t-il sur toutes les versions d'Excel ?

Oui, IMPUMAT est compatible avec toutes les versions d’Excel (Windows et Mac). Pour une compatibilité optimale, utilisez Office 2016 ou supérieur.

Est-il possible de modifier la mise en page ou les formules par erreur ?

Non, le fichier est protégé pour garantir la cohérence des calculs. Seules les zones de saisie sont modifiables.

Est-ce un paiement unique ?

Non, c’est un abonnement annuel avec accès aux mises à jour.

Comment vais-je recevoir IMPUMAT après l’achat ?

Vous recevrez un lien de téléchargement immédiatement après paiement via la marketplace sécurisée Payhip.
