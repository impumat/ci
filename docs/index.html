Frequently Asked Questions

Does IMPUMAT work without Internet?

Yes, the matrix runs autonomously in Excel, with no connection required.

Does IMPUMAT work with all versions of Excel?

Yes, IMPUMAT is compatible with all Excel versions (Windows and Mac). For optimal compatibility, we recommend Office 2016 or later.

Is it possible to accidentally change the layout or formulas?

No, the file is protected to ensure the accuracy of calculations. Only the input fields are editable.

Is this a one-time payment?

No, this is a yearly subscription with access to updates.

How will I receive IMPUMAT after purchase?

You will receive a secure download link immediately after payment through the Payhip marketplace.

Can I use IMPUMAT for both personal and professional accounting?

Yes, IMPUMAT is designed to be flexible. It can be used by freelancers, small businesses, and anyone who needs reliable accounting support in Excel.

Is there any technical support if I encounter difficulties?

Yes, our support team is available by email to assist you in case of questions or technical issues.
